# 🚀 Система Управления Банковскими Картами

## 📄 Описание проекта

**Backend-приложение на Java (Spring Boot)** для управления банковскими картами.
Система поддерживает **CRUD для карт**, **переводы между своими картами**, **маскирование номеров карт**, **аутентификацию/авторизацию с JWT** и **разделение прав доступа между ADMIN и USER**.

### 💳 Атрибуты карты

- **Номер карты**: зашифрован и отображается маской **** **** **** 1234  
- **Владелец**  
- **Срок действия** 
- **Статус**: Активна, Заблокирована, Истек срок  
- **Баланс**

### 🔑 Аутентификация и авторизация
- **Spring Security + JWT**
- **Роли**:
  - `ADMIN` — полный доступ
    - Создаёт, блокирует, активирует и удаляет карты  
    - Управляет пользователями  
    - Просматривает все карты и транзакции
  - `USER` — ограниченный доступ к своим картам
    - Просматривает свои карты  
    - Запрашивает блокировку карты  
    - Делает переводы между своими картами  
    - Проверяет баланс
   
### 🌐 API
- **CRUD операции для карт**  
- **Переводы между картами**  
- **Пагинация и фильтрация**  
- **Валидация запросов и сообщения об ошибках**

### 🔒 Безопасность
- **Шифрование данных**  
- **Ролевой доступ**  
- **Маскирование номеров карт**  

### 🗂 Работа с БД
- **PostgreSQL**  
- **Миграции через Liquibase** 

### 📝 Документация
- **Swagger UI**



## ⚙️ Установка и запуск

### 1. Клонируем репозиторий
```
git clone https://github.com/Bazu-Bazu/Bank-Card-Management-System
cd Bank-Card-Management-System
```

### 2. Сборка проекта с Maven
```
mvn clean install
```

### 3. Настройка Docker
```
docker-compose up -d
```
Поднимает базу данных и запускает приложение.

### 4. Пользование приложением
В браузере вводим:
```
http://localhost:8080/swagger-ui.html
```
Приложение готово к работе.
